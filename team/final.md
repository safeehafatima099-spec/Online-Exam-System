# Online Exam System

## Project Summary
This project is an Online Exam System developed using Data Structures concepts. It allows user registration, login, question management, exam attempt, result evaluation, and student deletion.

## Data Structures Used
- Linked List (User management, Question management)
- Queue (Delete student operation)

## Key Features
- Register and login users
- Add, delete, and display questions
- Attempt exam and calculate score
- Delete students using FIFO

## How to Run
- Compile and run the main file
- Follow menu options for operations
